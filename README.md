To-Do List Application

Description

This is a simple To-Do List application developed using Python. The application helps users manage their daily tasks efficiently. Users can add new tasks, view existing tasks, remove completed tasks, and exit the application.

Features

- Add Tasks
- View Tasks
- Remove Tasks
- Exit Application

Technologies Used

- Python

How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

python todo.py

Sample Menu

1. Add Task
2. View Tasks
3. Remove Task
4. Exit
